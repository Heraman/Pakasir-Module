# Pakasir PHP SDK (Unofficial)

PHP SDK sederhana untuk integrasi **Pakasir Payment Gateway** menggunakan REST API.  
Library ini membantu pembuatan payment URL, transaksi, simulasi pembayaran, pembatalan, dan pengecekan detail transaksi.
